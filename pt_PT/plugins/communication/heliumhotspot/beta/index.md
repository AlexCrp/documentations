# Plug-in de ponto de acesso de hélio

Este plugin permite monitorar a atividade de seus mineradores de hélio.

Os dados relatados pelo plugin são os seguintes :

* Escala de recompensa
* Mode
* Online
* Renda diária
* Renda desde o início
* Lista das últimas testemunhas
* Lista dos últimos testemunhados

Para configurar o plugin, basta indicar o endereço da sua carteira na configuração do plugin e depois clicar em "Sincronizar meus hotspots".
